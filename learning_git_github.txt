### Comandos ###

git status
git add
git commit -a -m '' -> commit
git push -> enviar para o github
git pull -> atualizar
git rm -> remover
git log -> Serve para acessar um histórico de modificações feitas no projeto.
git mv -> Serve tanto para mover quanto para renomear.
git checkout -> O arquivo modificado volta ao seu estado original.
git reset --hard origin/master -> todas as alterações commitadas e também pendentes serão excluídas.

.gitignore -> é um arquivo localizado na raiz do projeto -> podemos inserir todos arquivos
que não devem entrar no versionamento -> é útil para arquivos gerados automaticamente
ou arquivos que contêm informações sensíveis;

## Trabalhando com Branches ##
git branch -> Para visualizar os branchs disponíveis.
git branch algum_nome -> para criar uma nova branch.
'git branch -d' ou 'git branch --delete' -> serve para deletar uma branch.
